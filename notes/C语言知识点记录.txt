1.获取某个数据类型的长度可以使用 sizeof 操作符，如下所示：
int length = sizeof(变量名or变量类型);


2.进制:
	1) 二进制:
	二进制由 0 和 1 两个数字组成，使用时必须以0b或0B（不区分大小写）开头，例如：
	int a = 0b101;  //换算成十进制为 5
	int b = -0b110010;  //换算成十进制为 -50
	int c = 0B100001;  //换算成十进制为 33

	2) 八进制:
	八进制由 0~7 八个数字组成，使用时必须以0开头（注意是数字 0，不是字母 o），例如：
	int a = 015;  //换算成十进制为 13
	int b = -0101;  //换算成十进制为 -65
	int c = 0177777;  //换算成十进制为 65535

	3) 十六进制:
	十六进制由数字 0~9、字母 A~F 或 a~f（不区分大小写）组成，使用时必须以0x或0X（不区分大小写）开头，例如：
	int a = 0X2A;  //换算成十进制为 42
	int b = -0XA0;  //换算成十进制为 -160
	int c = 0xffff;  //换算成十进制为 65535
 			short	int		long
	八进制	%ho		%o		%lo
	十进制	%hd		%d		%ld
	十六进制	%hx %hX	%x %X	%lx %lX 
	(如果输入16进制的时候采用的是小写字母,则输出的时候也是小写,反之同理)
	注:在八进制与十六进制下没有负数,都为正数,也无法输入和读入负数.
	注: 推荐都是用小写字母.


3.无符号数unsigned:
	1).用法: 
	unsigned 变量类型 变量名
	注: 此时的变量则不具有正负之分,统一为正.
	2).输出方式:
	%u 以十进制形式输出无符号数；
	%o 以八进制形式输出无符号数；
	%x 以十六进制形式输出无符号数。
	即:整型的d换成u输出即可.


4.C语言中小数的指数形式为：
	1).表达: 
		aEn 或 aen
		a 为尾数部分，是一个十进制数；n 为指数部分，是一个十进制整数；
		E或e是固定的字符，用于分割尾数部分和指数部分。
		整个表达式等价于 a×10^n。
	2).输出:
	%e  以指数形式输出 double 类型，输出结果中的 e 小写；
	%E  以指数形式输出 double 类型，输出结果中的 E 大写；
	%le 以指数形式输出 double 类型，输出结果中的 e 小写；
	%lE 以指数形式输出 double 类型，输出结果中的 E 大写。

	特别的, 小数还有一种更加智能的输出方式，就是使用%g/%G。
	%g会对比小数的十进制形式和指数形式，以最短的方式来输出小数，
	让输出结果更加简练。所谓最短，就是输出结果占用最少的字符。
	注: %g默认最多保留六位有效数字，包括整数部分和小数部分；
		且%g不会在最后强加 0 来凑够有效数字的位数
		

5.转义字符:
	转义字符以\或者\x开头，以\开头表示后跟八进制形式的编码值，
	以\x开头表示后跟十六进制形式的编码值。
	对于转义字符来说，只能使用八进制或者十六进制。
	如: putchar('\61') <==> putchar(49) <==> putchar('\x31') 都是输出字符'1';


6.浮点取整函数: 
	floor(x) 返回的是小于或等于x的最大整数。
	ceil(x)  返回的是大于或等于x的最小整数。
	trunc(x) 返回的是x舍取小数位后的整数。(向0取整)
	round(x) 四舍五入
